# Simply-Data-Database-Creator
Create And Edit Encrypted / Non-Encrypted Sql3 Databases.
Designed Using Python 3.14.2 For Windows 11. 
English Or Spanish Languages Are Available In The File Menu.
Encrypted Databases Are Password Protected But Non-Encrypted Are Not.
If An Encrypted Database Is Created Under A Login Password, It Can Only 
Be Viewed By Logging In Under The Same Password. The Passwords Are Used 
Indirectly To Create a Encryption Key For The Creation Of A Database. 
Non-Encrypted Databases Can Be Viewed Under Any Login Password. 
Just Remember, If You Loose Your Password, You Loose Your Encrypted Database! 
The Login Password Can Be Changed At Any Time In The File Menu. 
Just Remember That The Last Password Change Will Become The Program Login Password.
