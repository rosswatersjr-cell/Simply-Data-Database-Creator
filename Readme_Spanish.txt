Cómo usar la base de datos Simply Data:

Los Menús Proporcionan Acceso Completo Para Crear, Editar, Agregar Y Eliminar.Además, Los Encabezados De Datos Y Las Celdas También Proporcionan Acceso Usando Clics Del Ratón.Un Clic Izquierdo En Cualquier Encabezado Permitirá Renombrar O Eliminar La Columna.Un Clic Izquierdo En Cualquier Celda De Datos Permitirá Editar Los Datos De La Celda.Un Clic Derecho En Cualquier Celda O Encabezado Producirá Una Suma De Esa Columna Si La Columna Está Definida Como Numérica, Real O Entera. Hacer Clic Izquierdo En Cualquier Número De ID En La Columna De ID Permite Editar O Eliminar Los Datos De La Fila Asociada.Los Datos De La Celda Pueden Ser De Cualquier Tipo Definido Excepto BLOG. Los Datos De La Celda Pueden Incluir Datos O Rutas A Sitios Web, Archivos De Música, Archivos De Video, Archivos De Imagen, Archivos PDF Y Documentos De Texto. La Ruta Completa Debe Ser Copiada A La Celda. Si Los Datos De La Celda Son Una Ruta, Un Clic Izquierdo Permitirá La Ejecución De Esa Ruta. Un Archivo Multimedia Abrirá Windows Media Player Y Reproducirá El Archivo Seleccionado. Windows Media Player Debe Estar Instalado. Si Tienes Un Visor De PDF, Se abrirá su visor y mostrará el archivo PDF. Los documentos de texto (.txt) abren el Bloc de notas de Windows para visualización. Además, otras bases de datos pueden ser importadas a Simply Data si no son bases de datos encriptadas. La base de datos original importada se deja intacta y solo se importa una copia de la original. A continuación, se presenta una lista de todas las extensiones admitidas para la opción de ruta de datos de celda y las extensiones aceptables para importar bases de datos. Todas las bases de datos creadas con este programa están encriptadas. La clave de encriptación para cada base de datos creada utiliza la contraseña de inicio de sesión para generar una clave de encriptación. Si se cambia la contraseña de inicio de sesión, las bases de datos creadas bajo un inicio de sesión diferente no estarán disponibles. Para acceder a las bases de datos creadas bajo diferentes inicios de sesión, simplemente restablezca la contraseña de inicio de sesión a la contraseña con la que se creó la base de datos. Si cambia las contraseñas, asegúrese de registrar todas las contraseñas pasadas y presentes.
