How To Use Simply Data Database:

The Menus Provide Full Access For Creating, Editing, Adding And Deleting.
Also, The Data Headings And Cells Also Provide Some Access Using Mouse Clicks.
A Left-Click On Any Heading Will Allow For Renaming Or Deleting The Column.
A Left-Click On Any Data Cell Will Allow For Editing The Cell Data.
A Right-Click On Any Cell Or Heading Will Produce A Summation Of That Column
If The Column Is Defined As Numeric, Real Or Integer. Left-Clicking Any ID Number
In The ID Column Allows For Editing Or Deleting The Associated Row Data.
Cell Data May Be Of Any Defined Types Except BLOG. Cell Data May Include Data Or Paths To 
Websites, Music Files, Video Files, Image Files, PDF Files And Text Documents. The
Complete Path Must Be Copied To The Cell. If The Cell Data Is A Path, A Left-Click
Will Allow For Execution Of That Path. A Multimedia File Will Open Windows Media Player
And Play The Selected File. Windows Media Player Must Be Installed. If You Have A PDF Viewer, 
A PDF File (.pdf) Will Open Your Viewer And Show The PDF File. Text Documents (.txt) Open
Windows Notepad For Viewing. Also, Other Databases May Be Imported Into Simply Data 
If They Are Not Encrypted Databases. The Original Imported Database Is Left Untouched
And Only A Copy Of The Original Is Imported. Below Is A List Of All Supported Extensions
For The Cell Data = Path Option And Acceptable Extensions For Importing Databases.
All Databases Created Using This Program Are Encrypted. The Encryption Key For Each
Created Database Uses The Login Password To Generate An Encryption Key. If The Login
Password Is Changed, Databases Created Under A Different Login Will Not Be Available.
To Access The Databases Created Under Different Logins, Simply Reset The Login Password
To The Password In Which The Database Was Created Under. If You Change Passwords, Make
Sure To Record All Past And Present Passwords.
